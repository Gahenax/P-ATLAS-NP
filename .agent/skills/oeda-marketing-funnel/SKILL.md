# OEDA Marketing Funnel & Pricing Strategy

No vendemos software de forma genérica. Vendemos la arquitectura para escalar y el derecho a poseer conocimiento matemático profundo.

Cuando estemos preparando una "Release" de producto (como un Dataset HDF5 o la carpeta `Mersenne-Gahen`), debes aplicar las siguientes 3 Capas de Distribución:

### Tier 1: The Research Note (Autoridad o Magnet)
- **Contenido:** PDF de reporte, indexación teórica, hallazgos clave.
- **Precio e Intención:** $19-$29 o Gratis (Pay what you want). Recolecta correos y demuestra que NO somos charlatanes. Se inyectan en sitios como SSRN.

### Tier 2: Demo Engine (Touching the Tech)
- **Contenido:** Tier 1 + Aplicación truncada o binario precompilado que opera localmente con datasets pequeños. Sin acceso modular a la matriz.
- **Precio e Intención:** $79-$149. "Ancla" visual y prueba fehaciente de capacidad operativa. Licencia restrictiva no-comercial.

### Tier 3: Premium B2B Engine (El Core Completo)
- **Contenido:** Repositorio inalterado. Archivos de Python puramente modulares, scripts completos Ouroboros, arquitectura escalable y licenciamiento comercial.
- **Precio e Intención:** $499 - $1,200+ USD. Está diseñado para que empresas asuman que pagar $1,000 es mucho más barato que emplear programadores durante 4 meses para replicarlo.
